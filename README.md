# Vue_JS 작업 공간

### npm 버전 확인
 : npm -v 

### nodejs 버전 확인
 : node -v

### global로 전체 설치 -> vue
 : npm install -g vue-cli

### vue 세팅하기
 :vue init webpack-simple todo_app

### setting

 cd todo_app
 //package.json에 있는 것 설치, maven으로 치면 pom.xml과 비슷
